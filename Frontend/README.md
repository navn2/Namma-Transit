# Ecolens Frontend Web Application

The frontend for Ecolens is a modern, responsive single-page web application designed to visually display real-time air quality, carbon metrics, heat indices, and exposure analytics.

## Technologies Used

*   **React** & **TypeScript**
*   **Vite**: Next-generation frontend tooling.
*   **TanStack Start & Router**: Full-stack framework with type-safe routing.
*   **MapLibre GL**: Fast, interactive vector maps.
*   **Tailwind CSS**: Utility-first CSS styling framework.

## Setup Instructions

### 1. Prerequisites
Ensure you have Node.js (version 18 or higher) and npm installed.

### 2. Install Dependencies
Run the following command in the frontend directory:
```bash
npm install
```

### 3. Environment Variables
Copy `.env.example` to `.env`:
```bash
copy .env.example .env
```

### 4. Running the App
Start the Vite development server:
```bash
npm run dev
```
The application will be available at [http://localhost:8080](http://localhost:8080).
